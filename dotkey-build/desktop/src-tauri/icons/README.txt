Place your app icons here:

- 32x32.png      - Windows taskbar
- 128x128.png    - Linux
- 128x128@2x.png - macOS Retina
- icon.icns      - macOS app icon (use iconutil or online converter)
- icon.ico       - Windows app icon

Generate from a 1024x1024 PNG using:
- macOS: https://github.com/abrignoni/iconutil or sips
- Windows: https://convertio.co/png-ico/
- Online: https://appicon.co/

Minimum viable: just put a 512x512 PNG named 128x128.png here for now.
